# 📊 Sales Analysis Project

This project focuses on analyzing retail sales data using Excel, SQL, and Power BI to generate meaningful business insights.

## 🔹 Tools Used
- Excel (Data cleaning and pivot analysis)
- SQL (Data querying and analysis)
- Power BI (Dashboard and visualization)

## 🔹 Project Workflow
- Cleaned and prepared raw data in Excel  
- Performed data analysis using SQL queries  
- Built an interactive dashboard in Power BI  

## 🔹 Key Insights
- Technology category generated the highest sales  
- West region recorded the highest profit  
- Sales show a steady increasing trend over time  

## 🔹 Conclusion
This project demonstrates an end-to-end data analysis process, from raw data to actionable insights using industry-standard tools.
